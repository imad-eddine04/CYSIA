## 1. Project Goal

The goal of this mini project is to build a small LAN in Packet Tracer and configure:
- Secure remote access to the router using **SSH**
- Network monitoring using **SNMP**
- Centralized logging using a **Syslog** server
- Time synchronization using **NTP**

Everything is done with one router, one switch, one PC (Admin) and one server.
